git init - Инициализация системы Git для текущей папки и созданение нового пустого репозитория*
ls -la - Содержание папки
git status - Текущее состояние репозитория
git add .  - Сохранение репозитория 
git commit -m "Название коммита" - Создание коммита
git log - История коммитов
git remote add origin "Ссылка на удаленный репозиторий" - Указываем удаленный репозиторий*
git push -u origin "Название ветки" - Отправка ветки на сервер
